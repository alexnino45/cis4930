BNPL Credit Risk Prediction
CIS4930 Pattern Recognition Final Project
Author: Alexander Nino
Predicting credit default risk using Point-of-Sale loan patterns as behavioral proxies for Buy-Now-Pay-Later activity.

The dataset is not included in this repository
Go to: https://www.kaggle.com/c/home-credit-default-risk/data and download
- application_train.csv (166 MB)
- POS_CASH_balance.csv (54 MB)

To run:
1. pip install -r requirements.txt
2. download 2 csvs
3. python bnpl.py
